# Settings

Access Settings via the gear icon in the top-right corner.

## LLM Providers

Configure API keys for Blackboard's AI analysis.

### Supported Providers

| Provider | Protocol | Default Model |
|----------|----------|---------------|
| Anthropic (Claude) | Anthropic | claude-haiku-4-5 |
| OpenAI | OpenAI | gpt-4o-mini |
| DeepSeek | OpenAI-compat | deepseek-chat |
| Groq | OpenAI-compat | llama-3.3-70b |
| DashScope (Alibaba) | OpenAI-compat | qwen-plus |
| Zhipu (GLM) | OpenAI-compat | glm-4-flash |
| Moonshot (Kimi) | OpenAI-compat | moonshot-v1-8k |
| MiniMax | OpenAI-compat | MiniMax-Text-01 |
| Volcengine (Doubao) | OpenAI-compat | doubao-1-5-pro-32k |
| Ollama (Local) | OpenAI-compat | llama3.2 |
| Custom | OpenAI/Anthropic | configurable |

### Import / Export

Memora supports SpeakOut JSON format for bulk import/export of API keys.

## Language

Switch between **English** and **中文**. Changes take effect immediately.

## Persona (L1)

View cross-project configuration files from your AI tools:
- Claude Code: `~/.claude/CLAUDE.md`, settings
- Codex: `~/.codex/AGENTS.md`, config
- Cursor: `~/.cursorrules`
